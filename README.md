## 📖 Overview

DevDeck is a customizable developer dashboard that combines GitHub, Vercel, project management, component prototyping, and AI-powered workflows into a single workspace.

Instead of switching between multiple tabs, DevDeck provides one centralized interface for monitoring repositories, deployments, sprint progress, and development tasks.

Designed with a clean, futuristic interface, DevDeck is built to be modular, scalable, and ready for future AI integrations.

## ✨ Features

### 📦 GitHub Integration
- View repositories
- Recent commits
- Repository activity
- Open issues
- Repository links

### 🚀 Vercel Dashboard
- Monitor deployments
- Deployment status
- Build history
- Deployment timestamps

### 🧪 Component Sandbox
- Live HTML/Tailwind preview
- Instant UI prototyping
- Interactive preview

### 📋 Sprint Board
- Drag-and-drop Kanban board
- Task management
- Multiple workflow stages

### ⚙️ Dashboard Customization
- Resizable widgets
- Hide/show widgets
- Drag-and-drop layout
- Sidebar navigation
- Dark mode

### 🔐 Secure Token Management
- GitHub Personal Access Token
- Vercel API Token
- Browser-only token storage
- Import/Export configuration
